# L'Asso

[L'Asso](https://lasso-education.up.railway.app/), is a non-profit association !

![L'Asso Logo](./public/images/common/L'Asso-382x379.png)

## Mission

We are a non-profit association committed to fighting educational inequality and expanding access to quality learning for all.

## What we do

We design and support open, inclusive educational initiatives for underserved communities.

## Why it matters

Education is a fundamental right and a key driver of social mobility, autonomy, and long-term impact.

## How we act

Through partnerships, open resources, and grassroots projects, we turn access into action.

## Get involved

Contributors, educators, and supporters are welcome to help us build fairer learning opportunities.

```bash
composer -v
composer init
composer dump-autoload

docker-compose down
# docker-compose build --no-cache
docker-compose up # then http://localhost:8080
```

- On Ubuntu, you could run the code as follows (once you have installed [LAMP tools](https://doc.ubuntu-fr.org/lamp))

```bash
sudo ./exec/run.sh # then start needed servers
```

- Then go at [http://localhost/lasso/](http://localhost/lasso/)
