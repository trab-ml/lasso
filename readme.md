# L'Asso

A non-profit association !

```bash
composer -v
composer init
composer dump-autoload

docker-compose down
docker-compose up # then http://localhost:8080
```

- On Ubuntu, you could run the code as follows (once you have installed [LAMP tools](https://doc.ubuntu-fr.org/lamp))

```bash
sudo ./exec/run.sh # then start needed servers
```

- Then go at [http://localhost/lasso/](http://localhost/lasso/)
